Main algorithm: Trust Region algorithm (TR) using Cauchy point
Other algorithms used: TR with dogleg method
Stopping criteria: error < tolerance

Problem to solve: min f(x)=ln(1+x'Ax), where A is a random, symmetric, positive definite matrix with specific eigenvalues, s.t.
  Optimal xmin: vector 0
  f(xmin) = 0

Main results: convergence plots in folder (.png files)

Main reference: Numerical Optimization by Nocedal
